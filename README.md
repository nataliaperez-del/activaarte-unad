# ACTiva Arte — UNAD

**ACTiva Arte** es una aplicación web de bienestar emocional basada en la Terapia de Aceptación y Compromiso (ACT), desarrollada para la comunidad universitaria de la UNAD.

## Descripción

Herramienta digital de acompañamiento emocional que integra recursos expresivos, reflexivos y de autorregulación emocional a través del arte y la escritura terapéutica.

## Tecnologías

- HTML5 semántico
- CSS3 con variables de diseño (design tokens)
- JavaScript vanilla (sin dependencias externas)
- Google Fonts: Nunito + Lora
- PWA-ready (offline support)
